# Secure Digital Wallet

A multi-platform, secure digital wallet application for payments and asset management.

## Features
- Web application (React)
- Mobile apps (iOS/Android)
- Secure backend API
- Cryptographic key management
- Transaction handling

## Project Structure
- `apps/web` - React web application
- `apps/mobile` - iOS and Android apps
- `packages/backend` - Node.js API server
- `packages/core` - Shared cryptographic libraries

## Getting Started

See deployment documentation in `docs/deployment.md`

## Technology Stack
- **Frontend**: React 18+
- **Backend**: Node.js with Express
- **Mobile**: React Native / Native Swift/Kotlin
- **Security**: Cryptographic libraries for key management
